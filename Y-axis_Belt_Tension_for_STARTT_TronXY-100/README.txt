# Summary

This is a Y-axis belt tensioner for STARTT printer. This is remixed based on the tensioner by Simhopp. This utilizes the hardware that comes with the printer so no need to buy extra. Print minimum 26% infill for the bracket as it gets a good cross pattern to ensure there is goos strength. The adjuster screw foot is printed 100% infill.


Note: This is printed with PLA

# Print Settings

Printer: STARTT
Rafts: No
Supports: No
Resolution: 0.2mm
Infill: 26% and 100%
